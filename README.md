# 영상 드래그앤드롭 변환기

PySide6 기반 Windows GUI 프로그램입니다. 영상 파일이나 폴더를 창에 드래그앤드롭하면 `ffprobe`로 분석한 뒤 `ffmpeg`로 순차 변환합니다.

## 기본 동작

- 기본 비디오 설정: `hevc_nvenc`
- 기본 프리셋: `p1`
- 기본 CQ: `28`
- 오디오: `copy`
- 출력 위치: `프로그램 실행 폴더`
- 출력 파일명: `원본파일명.hevc_nvenc_p1_cq28.mp4`
- 이미 HEVC인 파일은 기본적으로 건너뜁니다.
- 같은 이름의 출력 파일이 이미 있으면 기본적으로 건너뜁니다.
- 같은 세션에 같은 이름의 입력 파일이 여러 개 들어오면 출력 파일명 뒤에 `_2`, `_3`를 붙여 충돌을 피합니다.

## 실행 방법

1. `ffmpeg`와 `ffprobe`가 PATH에 있어야 합니다.
2. 의존성을 설치합니다.

```bash
python -m pip install -r requirements.txt
```

3. 앱을 실행합니다.

```bash
python app.py
```

## 사용 방법

1. 창을 실행합니다.
2. 영상 파일이나 폴더를 창에 드래그해서 놓습니다.
3. 기본값이면 자동으로 변환이 시작됩니다.
4. 진행률, 속도, 완료 여부는 표와 로그에서 확인합니다.

## 참고

- 폴더를 넣으면 재귀적으로 영상 파일을 찾습니다.
- 작업은 한 번에 하나씩 순차 처리합니다.
- 인코딩 중에는 CQ 값을 잠궈서 같은 배치가 섞이지 않게 했습니다.
- `PySide6`가 설치되어 있지 않으면 앱이 실행되지 않습니다.
